## Projetos

- [AwesomeMovieApp](AwesomeMovieApp/)

## Instalação

1. Abra o Android Studio.
2. Selecione **File > Open**.
3. Navegue até a pasta do projeto desejado (ex: `AwesomeMovieApp`) e selecione-a.
4. Aguarde o carregamento e sincronização do Gradle.
